# README – Pagina web per il download dei report di sostenibilità (Gruppo BF S.p.A.)

## 1. Scopo e ambito
Questo documento descrive come installare, verificare e mantenere la pagina statica che consente la consultazione e il download dei report di sostenibilità del Gruppo BF S.p.A. Il progetto è pensato per essere pubblicato su qualunque hosting statico e non richiede dipendenze esterne.
